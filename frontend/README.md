# 📊 Movie Analytics Dashboard (Frontend)

Este é o módulo de visualização de dados do projeto Movie Data Pipeline. Desenvolvido com **React + TypeScript**, o foco aqui é transformar dados complexos em decisões estratégicas para stakeholders do cinema.

## 💡 Funcionalidades Implementadas

- **Gráficos Dinâmicos:** Uso de Recharts com redimensionamento explícito via Hooks para garantir estabilidade em CSS Grids.
- **AI Insights Box:** Componente reativo que exibe análises estratégicas geradas por IA (Gemini/Groq), incluindo badge de verificação do provedor.
- **Real-time Logs:** Monitoramento da última execução do pipeline de dados integrado diretamente no header.

## 🛠️ Tech Stack

- **Vite:** Build tool ultra-rápida.
- **TypeScript:** Segurança de tipos em toda a aplicação.
- **Recharts:** Biblioteca de visualização de dados.
- **CSS Modules/Grid:** Layout responsivo e moderno.

## 🚀 Como rodar

1. `npm install`
2. `npm run dev`
